上傳後系統會用真人語音取代 TTS，對特教學生有更好的效果。
📋 預錄語音準備指引
🎤 錄音建議：找一位語調溫柔、咬字清晰的老師或配音者錄製
🐢 語速：比正常說話慢 20%，每句之間自然停頓 0.5–1 秒
😊 情感：正向鼓勵、不急促，答錯時語氣溫和不責備
🔊 格式：MP3 / WAV，建議 44.1kHz，單聲道即可，每個片段獨立一個檔案
📁 命名規則：建議依照下方片段名稱命名，方便對應

Firebase 連線設定
由明右提供 Firebase config，貼入後儲存即可連線資料庫

📋怎麼取得 Firebase Config？
1
請明右到 Firebase Console → 專案設定 → 你的應用程式 下方找到「firebaseConfig」物件
2
把整個 config 複製起來，看起來會像右邊的格式
3
貼到下方輸入區，按「儲存並連線」
4
點「測試連線」確認是否成功讀取資料庫
Config 格式範例
{
  apiKey: "AIzaSy...",
  authDomain: "your-project.firebaseapp.com",
  databaseURL: "https://your-project-default-rtdb.firebaseio.com",
  projectId: "your-project",
  storageBucket: "your-project.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:abc123"
}
🔧輸入 Firebase Config
將明右提供的 firebaseConfig 物件完整貼入，可以貼整段 JS 程式碼，系統會自動解析。
✅已儲存的連線設定
⚠ Config 儲存於瀏覽器 localStorage，換裝置或清除快取後需重新輸入。正式上線請由明右寫入程式碼。
